## QUICK REFERENCE: Build APK on Windows

### 🚀 FASTEST METHOD: Online Builder (5 min)
1. Go to https://buildozer.cloud/
2. Upload your project
3. Download APK

### ⭐ BEST METHOD: WSL 2 (40 min setup, then 3-5 min rebuilds)

**First time setup:**
```powershell
# PowerShell as Administrator
wsl --install
# Restart computer
```

**Then in Ubuntu terminal:**
```bash
cd /mnt/c/Users/PRASANNA/Desktop/"New folder"
bash build_apk_wsl.sh
```

**APK output:**
```
C:\Users\PRASANNA\Desktop\New folder\bin\smsspamdetector-1.0-debug.apk
```
